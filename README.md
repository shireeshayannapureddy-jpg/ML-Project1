# Titanic Survival Prediction App

## Project Description
This project performs Exploratory Data Analysis (EDA) and machine learning
on the Titanic dataset to predict passenger survival.
The application is built using Streamlit and deployed on Streamlit Community Cloud.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit

## How to Run Locally
1. Install dependencies:
   pip install -r requirements.txt
2. Run the app:
   streamlit run app.py

## Deployment
The app is deployed using Streamlit Community Cloud.

## Dataset
Titanic Dataset (CSV file)
